Conditions d'utilisation

Bienvenue sur le bot Discord associé à ce serveur ! En utilisant ce bot, vous acceptez de respecter les présentes conditions d'utilisation.

1. Utilisation du bot

Le bot offre plusieurs fonctionnalités, dont :

  • Commande wembed pour créer des messages enrichis (embeds) dans les canaux spécifiques.
  • Commande wbombe qui envoie des gifs animés dans certains canaux ou catégories autorisés.
  • Commandes wtriks et wfreetriks pour gérer temporairement le mute d’un utilisateur spécifique.
  • Lootboxes et récompenses virtuelles avec temporisation (cooldowns) pour chaque utilisateur.

2. Règles et limitations

  • Les commandes sont limitées par des cooldowns afin d'éviter les abus.
  • Les utilisateurs doivent respecter les canaux autorisés pour chaque commande.
  • Tout comportement abusif ou tentative de contourner les limites peut entraîner une suspension ou un blocage temporaire par le bot.

3. Responsabilité

Le bot est fourni tel quel. Les administrateurs ne sont pas responsables de :

  • Perte de récompenses virtuelles.
  • Problèmes techniques liés à Discord ou aux permissions.
  • Toute utilisation abusive ou non conforme du bot par les utilisateurs.

4. Protection des données

Le bot collecte uniquement les informations nécessaires pour fonctionner correctement :

  • ID d’utilisateur Discord pour suivre les cooldowns et les récompenses.
  • Références de messages pour les interactions avec les embeds.

Aucune donnée personnelle n'est vendue ou partagée avec des tiers.

5. Modifications des conditions

Les administrateurs peuvent modifier ces conditions à tout moment. Les utilisateurs seront censés accepter les conditions mises à jour en continuant d’utiliser le bot.

6. Contact

Pour toute question concernant ces conditions ou le bot, veuillez contacter un administrateur du serveur Discord.
